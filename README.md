# AWS EC2 Deployment Practice

This repository was used for practicing deployment of a Node.js application on AWS EC2.

## What I Practiced
- Launching EC2 instance
- Connecting through SSH
- Linux package installation
- Git & GitHub workflow
- Node.js application setup
- Security Group configuration
- Running application on port 3000

## Tech Stack
- AWS EC2
- Ubuntu Linux
- Node.js
- Git & GitHub

## Deployment Steps
1. Launch EC2 instance
2. Connect using SSH
3. Install Node.js and npm
4. Clone repository
5. Configure environment variables
6. Run application using npm

## Note
This project was used only for deployment and DevOps practice purposes.
